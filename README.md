# PrivateRepo

GitHUB repository for most of my scripts.

## Getting Started

Just use most scripts "as is" - sorted by folders.

### Prerequisites

```
PowerShell
```

and/or

```
PowerCLI
```

### Installing

#### Powershell ####

Comes preinstalled with Windows 10

#### PowerCLI ####

```Install-Module -Name VMware.PowerCLI ```
